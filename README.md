# tour_de_maroc
Plateforme digitale pour le "Tour de Maroc", permettant aux fans de suivre les étapes, aux cyclistes de partager leurs parcours, et aux organisateurs de gérer l'événement efficacement.

``` 
    public/
        assets/
            images/
        css/
        components/
            header.twig
            footer.twig
            ...
        index.php
        .htaccess

    .../
```
